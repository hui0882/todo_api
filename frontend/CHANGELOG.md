# 更新日志

## 2026-02-12

### 优化
- **API 配置页面无需登录即可访问**
  - 在登录页面右上角添加了"API 配置"按钮
  - 用户可以在登录前配置后端地址
  - 避免了因后端地址错误导致无法登录的问题
  - 配置页面会根据登录状态显示不同的导航按钮（"去登录" 或 "返回首页"）

### 功能
- 登录/注册页面新增"API 配置"快捷入口
- 设置页面增加友好提示，说明可在登录前使用
- 优化了页面导航逻辑，提升用户体验

### 使用建议
如果你的后端地址不是默认的 `http://localhost:8080`，建议：
1. 访问前端后，先点击"API 配置"
2. 配置正确的后端地址
3. 测试连接确保配置正确
4. 保存后再进行登录操作
